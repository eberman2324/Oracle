1. Switch 3 users to TEMP_TEMP

ALTER USER AETNADBA
 TEMPORARY TABLESPACE TEMP_TEMP

/
ALTER USER MDSYS TEMPORARY TABLESPACE TEMP_TEMP

/
ALTER USER ORACLE TEMPORARY TABLESPACE TEMP_TEMP

/


2. Drop Temp2

CREATE TEMPORARY TABLESPACE TEMP2
    TEMPFILE '/oratemp/ppmu300/PPMDEBIZ/temp02.dbf' SIZE 10240M AUTOEXTEND OFF
    TABLESPACE GROUP TEMP
    EXTENT MANAGEMENT LOCAL UNIFORM SIZE 1M DEFAULT NO INMEMORY
/

DROP TABLESPACE TEMP2 INCLUDING CONTENTS AND DATAFILES CASCADE CONSTRAINTS

/

3. Drop Temp1

CREATE TEMPORARY TABLESPACE TEMP1
    TEMPFILE '/oratemp/ppmu300/PPMDEBIZ/temp01.dbf' SIZE 10240M AUTOEXTEND OFF
    TABLESPACE GROUP TEMP
    EXTENT MANAGEMENT LOCAL UNIFORM SIZE 1M DEFAULT NO INMEMORY
/

DROP TABLESPACE TEMP1 INCLUDING CONTENTS AND DATAFILES CASCADE CONSTRAINTS

/

4. Create Temp

CREATE TEMPORARY TABLESPACE TEMP
    TEMPFILE '/oratemp/ppmu300/PPMDEBIZ/temp.dbf' SIZE 10240M AUTOEXTEND OFF
    TABLESPACE GROUP TEMP
    EXTENT MANAGEMENT LOCAL UNIFORM SIZE 1M DEFAULT NO INMEMORY
/

5. Switch 3 Users to Temp

ALTER USER AETNADBA
 TEMPORARY TABLESPACE TEMP

/
ALTER USER MDSYS TEMPORARY TABLESPACE TEMP
/
ALTER USER ORACLE TEMPORARY TABLESPACE TEMP
/

CREATE TEMPORARY TABLESPACE TEMP_TEMP
