1. Install SQLT package if not already there
Download from oracle SQLT_installation_logs_archive.zip for correct Oracle Version

2. Unzip into your local PC that will create following dir
sqlt

3. FTP/SFTP this into Oracle Box


Follow sqlt_instructions.html for more instructions.

4. Install

cd sqlt/install
 sqlplus / as sysdba
 START sqcreate.sql

Password for user SQLTXPLAIN: Locked#99999
Tbls: AUDIT_SPACE
Temp Tbls: TEMP
Main application user of SQLT: PROD
Oracle Pack license [T]: T

cd /home/oracle/eb/sqlt/run
